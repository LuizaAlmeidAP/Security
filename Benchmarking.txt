Benchmarking basics:

For the benchmarking of the project we use the following rules:
Disable CPU turbo boost/power-saving mode;
Avoid having background processes running;
Warm-up for more consistent results;
Running multiple times and averaging the result
Using "pyperf" to achive the result python library (https://pyperf.readthedocs.io/en)


AES benchmarking: 
For AES we measure 
The throughput(Data per second);
The Mean Encription/Decription time ;
The Standard deviation;
The Median;


RSA benchmarking:

For RSA we measure
the number of public/private-key operations(key generation) per second;
The Encription/Decription time;
The Throughput;


SHA-256: benchmarking:
The Throughput;
The Time per hash:;
